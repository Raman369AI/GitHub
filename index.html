End-to-End Predictive Modeling for Mushroom Toxicity Classification
  • Objective: Developed a comprehensive, end-to-end solution to identify toxic mushroom species.
  • Process: Conducted data preprocessing, feature engineering, and exploratory data analysis (EDA).
    Built and fine tuned an ensemble model using LightGBM, achieving a 98% AUC.
  • Deployment: Developed and deployed a FastAPI-based web service on Azure App Service with Docker for scalability, incorporating a Streamlit app for real-time classifications.
  • Tech Stack: LightGBM, FastAPI, Azure App Service, Docker, Streamlit.
